# 业务指标定义文档

## 销售额
销售额 = SUM(order_items.subtotal) 或 SUM(sales_orders.total_amount - sales_orders.discount_amount)
按时间维度可分：日销售额、周销售额、月销售额、季度销售额
按维度可分：门店销售额、品类销售额、商品销售额
销售额环比 = (本期销售额 - 上期销售额) / 上期销售额 * 100%
销售额同比 = (本期销售额 - 去年同期销售额) / 去年同期销售额 * 100%

## 坪效
坪效 = 门店销售额 / 门店面积(area_sqm)
坪效反映单位面积产出效率，是零售行业核心指标
坪效越高说明门店空间利用率越好

## 库存周转率
库存周转率 = 销售成本 / 平均库存
平均库存 = (期初库存 + 期末库存) / 2
库存周转天数 = 365 / 库存周转率
周转率越高说明库存管理效率越好

## 毛利率
毛利率 = 毛利润 / 总收入 * 100%
毛利润 = 总收入(total_revenue) - 总成本(total_cost)
毛利率反映产品盈利能力，是财务核心指标
不同品类毛利率差异较大

## 客单价
客单价 = 总销售额 / 订单数量
客单价反映单笔交易的平均消费水平
提升客单价是增长策略之一

## 会员贡献率
会员贡献率 = 会员消费总额 / 全部消费总额 * 100%
反映会员体系对整体销售的贡献
高会员贡献率说明会员体系运营良好

## 库存预警
当 stock_quantity < safety_stock 时触发库存预警
需要及时补货以避免缺货损失
库存预警率 = 预警商品数 / 总商品数 * 100%

## 同比环比
环比 = (本期值 - 上期值) / 上期值 * 100%
同比 = (本期值 - 去年同期值) / 去年同期值 * 100%
环比反映短期趋势变化，同比消除季节性影响
