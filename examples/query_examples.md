# 查询示例

以下是 TextSQL-Agent 支持的典型查询场景，基于零售测试数据集。

## 销售分析

### 基础查询
```
上月各门店销售额排名
```
```
本周每日销售额趋势
```
```
TOP10 畅销商品列表
```

### 对比分析
```
本月与上月销售额对比
```
```
各品类销售占比
```

## 库存分析

```
当前库存周转率最低的10个商品
```
```
库存预警商品清单（库存<安全库存）
```
```
各仓库库存金额统计
```

## 客户分析

```
会员复购率统计
```
```
客户消费等级分布
```
```
新客增长趋势
```

## 财务分析

```
本月毛利率分析
```
```
各品类利润贡献
```
```
坪效排名（按门店）
```

## 复杂查询

```
对比Q1和Q2各门店销售额变化，找出增长最快的3家门店
```
```
分析上月滞销商品，给出库存优化建议
```
```
按客单价分层统计用户数量和贡献销售额
```
